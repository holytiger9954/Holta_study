Eclipse existing project structure guide

This package is arranged to match a project that uses:
- src/main/java
- src/main/webapp

How to apply into your existing Eclipse project:
1. Back up your current project.
2. Copy the contents of src/main/java into your project's src/main/java.
3. Copy the contents of src/main/webapp into your project's src/main/webapp.
4. Refresh the project in Eclipse (F5).
5. Make sure Apache Tomcat 9 is selected in Targeted Runtimes.
6. Make sure Java build path includes your src/main/java source folder.
7. Run on Server.
